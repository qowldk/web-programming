# 🥕당근북클럽📖
매 학기 초만 되면 강의 책을 좀 더 싸게 사고 싶어 하는 학생이 있고 방에 가득 쌓여 있는 강의 책을 되팔아 용돈을 벌고 싶어 하는 학생도 있습니다. 이러한 학생들을 서로 연결해 주는 플랫폼이 부족하다는 생각에서 시작해 책을 위주로 거래하는 “당근북클럽”이라는 웹사이트를 제작하게 되었습니다.
